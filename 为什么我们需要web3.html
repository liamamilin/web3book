The key of web3

Chapter 2 为什么我们需要web3

互联网：未曾成熟却已老去

要理解 Web 3.0 的诞生动因，我们需要回顾互联网的发展历程和现状。

互联网的兴起奠定了人类迈入信息社会的基石。一方面，它极大降低了信息记录的成本；另一方面，只有当信息总量达到一定规模后，才能对社会和经济活动产生显著影响。

在上世纪 90 年代，互联网的诞生为信息传播打开了新的渠道，孕育了一批专业内容生产平台（PGC，Professionally Generated Content），如一些以专门化内容为核心的平台。然而，这一时期的互联网是单向的，用户仅能被动接收信息，而数据的产生与用户几乎无关——这是 Web 1.0 时代。

随着时间推移，用户逐渐不满足“只能输入而无法输出”的状态，渴望表达的需求催生了一大批互动性平台，如 Facebook（现 Meta）和 Twitter。同时，电商平台、视频网站等也纷纷涌现，为用户提供参与和互动的机会。这一阶段标志着互联网进入 Web 2.0 时代。

2.1 互联网公司的主宰：服务的强大与用户的无力

在 Web 2.0 时代，用户的行为数据成为互联网公司的核心资产。然而，这些数据被“存储即拥有”，而非“谁创造谁拥有”。这种模式使互联网公司凭借数据资源迅速壮大，同时也引发了与公众利益的深刻冲突。

举例来说，2021 年，Meta（原 Facebook）的市值曾一度突破 1 万亿美元，相当于全球 GDP 排名的前 15 名。拥有 19 亿日活用户和 29 亿月活用户，Meta 的平台已深入全球 36% 的人口生活。然而，这种规模的垄断意味着用户的网络行为完全受控于平台规则。例如，用户发送消息或发表观点必须遵循平台的政策，否则可能面临“社交性死亡”，甚至像特朗普一样遭遇禁言。

互联网公司的控制不仅限于社交领域。例如，在支付领域，用户的一笔转账必须通过金融机构完成，如果遇上非营业时间，甚至可能无法执行。而即便一些机构提供免费的服务，它们的背后仍隐藏着广告或贷款推荐等商业利益。

这意味着，虽然互联网服务看似赋予用户极大便利，但用户实际上被互联网巨头“绑架”，在这些公司精心设计的制度面前毫无议价空间。互联网变得越来越强大，但也越来越像一头“灰犀牛”，将私人利益强加为社会规则。

2.2 被困的互联网：自由空间的需求

近年来，互联网巨头的主宰不仅带来了经济利益的不平衡，也引发了一系列社会问题。

一方面， 数据泄露与隐私问题 越来越严重，网络暴力和数据滥用对用户生活的负面影响不容忽视。例如，2022 年，Facebook 承认其用户数据被剑桥分析公司非法使用，影响了 5000 万用户，并被用于特朗普的竞选活动。

另一方面，互联网公司的行为可能直接与国家利益产生冲突。2021 年，澳大利亚政府推出《新闻媒体和数字平台强制议价规则》，要求互联网平台为新闻内容支付费用。然而，Facebook 不愿接受这一规则，直接屏蔽了澳大利亚政府及机构的所有信息发布功能，包括疫情防控等紧急信息。这一事件凸显了私人公司对公共利益的严重威胁。

互联网公司通过提供服务积累了巨大的社会影响力，但当私人利益凌驾于公共利益之上时，市场失灵的弊端就显现无遗。互联网的发展急需一个既能平衡商业属性与社会属性，又能为创新提供空间的新起点。

2.3 Web 3.0：一场自下而上的互联网革命

Web 3.0 的使命正是解决当前互联网的局限性。它不仅是一次技术架构的开放性重塑，更是一场旨在改变互联网“精神气质”的革命。

目前，Web 3.0 的定义尚未完全统一，但以太坊前 CTO Gavin Wood 将其描述为“通过区块链和去中心化协议支持的新一代互联网”，这一描述为我们提供了清晰的方向。Web 3.0 的核心是 功能协议化，这些协议没有更多的利益诉求，仅按照规则运行，通过自由组合实现协作与创新。

例如，Web 2.0 中的支付平台（如支付宝）承担了交易担保的角色，但同时绑定了借贷等商业模式，对市场竞争产生了抑制。而在 Web 3.0 中，这一功能将由“哈希时间锁协议”（HTLC）这样的去中心化协议完成，消除了对单一平台的依赖，为市场带来更公平的竞争环境。

2.4 Web 3.0 的“全栈”架构：从硬件到经济系统

Web 3.0 的实现需要两大支柱：硬件系统 和 经济系统。

硬件系统
Web 3.0 的硬件基础是分布式技术架构，通过通信、计算、存储和交互等层面的协议化设计，消除了对单一机构的依赖。这种架构的特性被称为 “去信任的信任”（Trustless Trust）。例如，基于区块链共识的 IC 协议通过虚拟子网实现了数据计算的高效协作，克服了传统架构中的单点故障问题。

经济系统
硬件之外，Web 3.0 需要一套围绕用户身份认证、数据记录、授权与激励的经济系统。区块链为这套系统提供了基础，通过代币机制激励参与者，协调协议间的利益分配。这种经济系统不仅实现了协作的高效性，也创造了更多的创新机会。

2.5 Web 3.0：数据创造、拥有与激励的新纪元

虽然从用户体验上看，Web 3.0 可能与 Web 2.0 并无太大区别，但它背后的技术逻辑和商业模式却完全不同。Web 3.0 将通过分布式技术和经济系统的结合，打造一个高度连接、无国界的数字经济体。这场变革赋予了用户对数据的创造、拥有和激励权，为我们带来了全新的产业机会和创新空间。

Web 3.0 是一次“破坏性创新”，它不仅重塑了互联网的架构，还为数字经济注入了新的生命力。对于这片新大陆，行动比心动更重要。探索 Web 3.0 的机会，不仅是技术发展的趋势，更可能是每个人成为历史参与者的一次重要契机。
